# RNS-C: Embedded Reticulum Network Stack

A lightweight, high-performance C++ implementation of the [Reticulum Network Stack](https://reticulum.network/) designed for the ESP32.

## Features
* **Native LoRa**: SX1262 (Heltec V3).
* **Multi-Interface**: Routes packets between LoRa, BLE, WiFi, and Serial.
* **Store & Forward**: Saves packets to Flash.
* **Sideband Support**: Works with the Android App via BLE.
* **Computer Modem**: Works with `nomadnet` via USB.

## Build
1. Install VS Code & PlatformIO.
2. Open this folder.
3. Click Upload.
